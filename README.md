# Neon Drive – Retrowave Racer

Retro-inspirierter Endless-Runner im Synthwave-Look. Du steuerst einen Neon-Flitzer durch den Nachthimmel, weichst entgegenkommendem Verkehr aus und jagst deinem Highscore hinterher.

## Spielen

1. Öffne `index.html` in einem modernen Browser.
2. Drücke **SPACE**, um das Spiel zu starten.
3. Steuere mit den Pfeiltasten oder `A`/`D`. Halte **SHIFT**, um kurzzeitig zu boosten.
4. Weiche den anderen Autos aus, sammle Punkte und knacke deinen Bestwert.

Das Spiel speichert deinen Highscore lokal im Browser (LocalStorage).
